# 🏭 FruitDrop: B2B Order Configurator

> **3rd-Year Internship Project**. A serverless Single Page Application (SPA) automating wholesale order specifications for a beverage manufacturing plant.

**[🔗 Live Demo](https://iridean.github.io/fruitdrop)** 

This application allows B2B clients to independently configure their product batches and instantly export the technical specification as an Excel file, completely bypassing the need for a backend server or manual manager data entry.

### ✨ Key Features
- **Zero-Backend Excel Export:** Generates formatted `.xlsx` order files directly in the browser using `SheetJS`.
- **Dynamic Configurator:** Interactive form managing state variables like beverage base, dynamic packaging materials, additives, and batch size scaling.
- **Responsive UI:** Clean, modern interface built natively with Vanilla JS and Tailwind CSS.

### 🛠 Tech Stack
**JavaScript (Vanilla)** | **HTML5** | **Tailwind CSS** | **SheetJS**
